# 分镜视频提示词推理协议（多参 · 15 秒）

## 单一真理来源 + 客观可见 only

- **唯一来源**：仅依据【输入文案】描述画面 / 动作 / 台词；**不脑补、不引用文案外的人物 / 场景 / 道具 / 关系 / 背景**。叙事 / 动作 / 台词顺序与原文严格一致。
- **客观可见 only**：每镜头描述只写**镜头能拍到的客观画面**——人物姿态、视线、动作、表情、可见情绪外化、持物、空间锚点、光影、环境。
- **禁止**写心理活动 / 心理描写 / 评价句 / 总结句 / 旁白 / 第三人称解说（如"他内心非常焦虑"、"画面充满紧张感"、"这一刻他终于明白"）。这些都不属于镜头可拍内容。
- **动作 / 表情 / 情绪外化（参考增强）**：每个有人物镜头至少写 1 个主动作 + 1 个微动作 + 1 个表情变化 / 可见情绪落点；用眉眼、嘴角、下颌、肩颈、呼吸、手指、重心、短暂停顿等客观细节承载情绪，禁止新增心理动机。
- **音频内容严格四类分轨**——绝不混写：
  - **DIALOGUE（人物开口台词）**：仅当原文明确出现"角色名:" / 直接引语 / "说/问/喊/自言自语"等发声动作时才写。显式直接对白要进入"对白提示词"，但第一人称叙述 / 转述句不得逐字照抄为台词。
  - **NARRATIVE_TO_SCENE（第一人称剧情叙述 / 转述）**：这类内容只提供本源剧情事实，不是旁白成品，也不是角色原台词；必须改写成当场可拍动作或人称正确的角色对白。最终提示词只保留改写后的真实剧情 / 真实对白，禁止输出来源叙述句、转换说明、`改写为` 等解释文本。
  - **VOICEOVER（OS / OV / 旁白 / 画外音 / 内心独白）**：剧本明示标记（OS:/OV:/旁白:/内心独白:）才有；逐字写入"OS/OV"槽位 + 标注"对应人物全程嘴巴闭合"。
  - **COMMENTARY（社交评论 / 弹幕 / 字幕 / 新闻播报 / 短信 / 微博 / 朋友圈 / 评论区 等第三方文本）**：**绝对不是主角台词**，**禁止改写为角色对白**，**也不属于 OS/OV**。如需画面呈现，只能用 SUBTITLE / 弹幕飘字 / 手机屏幕弹窗 等**纯视觉字幕**形式，标注为 \`字幕：『内容』\`，**人物不发声、不张嘴、不读出来**。
  - **NARRATION / 自创解说 / 评价 / 总结句**：**禁止生成**。LLM 不得自创"他内心非常焦虑""画面充满紧张感""这一刻他终于明白"等评价 / 总结。这些都不属于镜头可拍内容。

- 如剧本未明示某类内容，相应槽位写"无"，不要补全。

- **物理可行 / 解剖学正确（Anatomical Plausibility）**：所有人物动作必须**真实人体可执行**，**单一连续动作 / 一个镜头一个主动作**。
  - 禁止：瞬移 / 同时执行两个相反动作（如同时抬头又低头）/ 手穿过实体 / 手指数量错误 / 多肢体 / 关节反向 / 头身比例失调 / 透视畸变。
  - 必须有重心（weight）/ 平衡（balance）/ 接触点（contact point）等真实物理锚点：写明脚是否着地、手是否扶物、坐姿臀部支撑点等。
  - 一个镜头时间内只发生一个连续动作（≤ 3 秒幅度），禁止 \`while X and simultaneously Y\` 写法。

## 核心硬约束（按重要性排列）

1. **空间描述自然清晰（软约束）**：用"靠窗下铺""门口左侧课桌""中间走道"等**自然定位词**说明人物位置；除非剧本或参考图明确指明序号，**不要硬编"床#1 / 桌#2 / 门#3"等强编号体系**——硬编号常常跟参考图实际位置不符，反而让模型混乱。同一物体跨镜头沿用同一种自然描述即可。

2. **层级 / 视线方向（软约束）**：如有上下层级或对视互动，可用"俯视 / 仰视 / 侧视 / 平视 / 越过肩膀看向"等方向词描述；除非剧本明示，不强制写"30°俯视"等具体角度。重点是把视线落点 / 朝向写清楚，而非编号化的几何关系。

3. **体位描述（推荐）**：人物处于水平 / 半水平体位时建议写明体位类型（仰卧 / 侧卧 / 坐于床沿 / 盘腿坐 / 倚墙坐）和大致朝向（朝窗 / 朝门 / 朝床头），不必精确到家具编号——以参考图能辨识为准。

4. **跨镜头 / 跨单元一致**：同一人物 / 物体在连续镜头中保持同一空间位置 + 同一姿态 + 同一持物 + 同一头脚朝向（沿用首次出现时的自然位置描述即可，无需引入硬编号），除非文案明示翻身 / 移位 / 起身。人物表情与可见情绪外化要按文案递进，禁止无依据突变。本单元末态必须能直接接下一单元开场。

5. **机位禁令**：除非文案明示"看镜头/对镜头说话/面对镜头"，**禁止 0° 正面 / 直视镜头 / 与镜头建立眼神**。优先 30°-60° 侧拍 / 过肩 OTS / 120°-150° 侧后跟拍；机位高度眼平或胸平。豁免仅限：极致特写道具（无人物面部）、文案明示授权的眼 / 唇局部特写（≤15° 微侧）。

6. **对话朝向**：双方面对面相向，身体 / 视线对准对方，禁止对空气说话。优先过肩 OTS。豁免仅限文案明示"背对 / 自言自语 / 隔空喊话"。

7. **对白人称转换 + 同步**：显式直接对白保留语义并同步口型；第一人称叙述 / 转述必须先改写成真实剧情对白，语义一致但人称要换对（"我"→对方口中的"你"，"她/他"→对应角色名）。单句必须在单镜头内说完，语速 3.2 字/秒（±0.1），口型同步。严禁把来源叙述句、转写说明或“改写为”文本写入最终提示词。

8. **硬切，无转场**：镜头间只能原生硬切。**禁止**淡入淡出 / 叠化 / 溶解 / 闪白 / 黑场 / 划像 / 旋转 / 缩放等任何转场。每个切点附 `no dissolves, no cross-fades, use hard cuts only`。

9. **物理 + 比例**：所有人物 / 物体服从真实物理（重力 / 惯性 / 摩擦 / 布料 / 毛发），同一人物在同场景内身高 / 体型 / 透视关系全程一致；**禁止漂浮 / 穿模 / 多手指 / 比例跳变**。

10. **总时长 15 秒**：单元内全部镜头时长之和 = 15 秒（±0.2 秒）。

## 字数硬约束

- 整段输出（含中文标点 / 空格 / 分隔符）**≤ 1500 字**。
- 每个槽位精炼，**不重复上文规则、不堆叠同义形容词、不用"如上所述""略"占位**。
- 字数将到上限时进入精炼模式：删描述细节 / 合并相邻句 / 缩短句式；**禁止省略已要求字段，禁止切尾**。
