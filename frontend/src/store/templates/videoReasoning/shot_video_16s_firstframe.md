# 分镜视频提示词推理协议（首帧延展 · 16 秒）

## 单一真理来源 + 客观可见 only

- **唯一来源**：仅依据【输入文案】描述画面 / 动作 / 台词；**不脑补、不引用文案外的人物 / 场景 / 道具 / 关系 / 背景**。
- **客观可见 only**：描述只写**镜头能拍到的客观画面**——人物姿态、视线、动作、表情、持物、空间锚点、光影、环境。
- **禁止**写心理活动 / 心理描写 / 评价句 / 总结句 / 旁白 / 第三人称解说。这些都不属于镜头可拍内容。
- **音频内容严格四类分轨**——绝不混写：
  - **DIALOGUE（人物开口台词）**：仅当原文明确出现"角色名:" / 直接引语 / 发声动作时才写。显式直接对白要进入"对白提示词"，但第一人称叙述 / 转述句不得逐字照抄为台词。
  - **NARRATIVE_TO_SCENE（第一人称剧情叙述 / 转述）**：这类内容只提供本源剧情事实，不是旁白成品，也不是角色原台词；必须改写成当场可拍动作或人称正确的角色对白。最终提示词只保留改写后的真实剧情 / 真实对白，禁止输出来源叙述句、转换说明、`改写为` 等解释文本。
  - **VOICEOVER（OS / OV / 旁白 / 画外音 / 内心独白）**：剧本明示标记才写；逐字 + 标注"对应人物全程嘴巴闭合"。
  - **COMMENTARY（社交评论 / 弹幕 / 字幕 / 新闻播报 / 短信 / 微博 等第三方文本）**：**绝对不是主角台词**，**禁止作为角色对白**。如需画面呈现，仅用屏幕字幕 / 弹幕飘字 / 手机屏幕弹窗等纯视觉字幕形式，标注为 \`字幕：『内容』\`，**人物不发声、不张嘴、不读出来**。
  - **NARRATION / 自创解说 / 评价 / 总结句**：**禁止生成**。

- **物理可行 / 解剖学正确（Anatomical Plausibility）**：所有人物动作必须**真实人体可执行**，**单一连续动作**。禁止瞬移 / 手穿实体 / 手指数量错误 / 多肢体 / 关节反向 / 同时两个相反动作。必须明确重心 / 平衡 / 接触点（脚是否着地、手是否扶物、坐姿支撑点）。

## 核心硬约束（按重要性排列）

1. **首帧本质**：单图必须是当前镜头 **0 秒承接帧 / 动作起手帧 / 情绪稳定帧**——不是高潮帧、不是结果帧、不是海报图。

2. **空间描述自然清晰（软约束）**：用"靠窗下铺""门口左侧课桌"等**自然定位词**说明人物位置；除非剧本或参考图明确指明序号，**不要硬编"床#1 / 桌#2"等强编号体系**——硬编号常常跟参考图实际位置不符。

3. **层级 / 视线方向（软约束）**：如有上下层级或对视互动，可用"俯视 / 仰视 / 侧视 / 平视 / 越过肩膀看向"等方向词描述；除非剧本明示，不强制写具体角度。重点是视线落点 / 朝向清晰。

4. **体位描述（推荐）**：人物处于水平 / 半水平体位时建议写明体位类型 + 大致朝向（参考图能辨识即可），不必精确到家具编号。

5. **跨分镜一致**：人物外观 / 服装 / 持物 / 站位区 / 朝向严格继承上镜头末态；除非文案明示，禁止突变。本镜头末 0.3-0.5 秒必须为下镜头预留稳定衔接态。

6. **角色继承**：人物外观（年龄 / 发色 / 发型 / 体型 / 穿着 / 妆造 / 武器）**100% 继承已绑定的角色图，不在提示词中重复展开**。提示词只负责锁定身份 / 数量 / 站位 / 朝向 / 视线 / 说话状态 / 动作起手态 / 景别 / 机位 / 光影 / 特效 / 音频。

7. **机位 / 直视镜头禁令**：除非文案明示"看镜头/对镜头说话"，**禁止 0° 正面 / 直视镜头**；优先 30°-60° 侧拍 / 过肩 OTS / 侧后跟拍。OS/OV 对应人物全程闭嘴、不享有直面授权。

8. **16 秒视频规则**：
   - 0 秒 = 单图首帧；16 秒 = 结束帧；全程无切镜 / 无转场 / 无废帧。
   - 视频全程**唯一视觉依据是本次单图** + references；只允许在单图基础上做微幅动态延展。
   - 镜头机位 / 景别 / 轴线全程固定（除非文案明示运镜，否则不允许移动）。
   - 人物重心位移 ≤ 肩宽 10%，尺寸变化 ≤ 8%，脚底接地点不离开首帧位置；**禁止走 / 跑 / 换位 / 进出画 / 突然转身 / 大幅动作**（除非文案明示）。
   - 默认动作白名单：呼吸起伏 / 眨眼 / 眼神变化 / 眉部微动 / 发丝衣角轻摆 / 手指微动 / 持物轻颤 / 粒子光效。

9. **对白人称转换 + OS/OV**：显式直接对白保留语义并同步口型；第一人称叙述 / 转述必须先改写成真实剧情对白，语义一致但人称要换对（"我"→对方口中的"你"，"她/他"→对应角色名）。OS/OV 只在原文明确标记时写入，且对应人物全程嘴巴完全闭合。

10. **音频规则**：去除所有背景 BGM；仅保留台词 / OS/OV / 文案明示的环境音 + 动作音。

## 字数硬约束

- 图片段（[图片提示词]） **≤ 600 字**；视频段（[视频提示词]）**≤ 800 字**；分隔符 / 空行 ≤ 100 字；合计 **≤ 1500 字**。
- 每个槽位精炼，禁堆叠同义形容词、禁"如上所述"、禁"略"占位。
- 字数将到上限进入精炼模式：删细节 / 合并句式；**禁止省略已要求字段，禁止切尾**。
